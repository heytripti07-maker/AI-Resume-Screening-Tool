# AI Resume Screening Tool

# Description
This project compares a candidate's resume with a job description using TF-IDF and Cosine Similarity to calculate the matching score.

# Technologies Used
- Python
- Streamlit
- Scikit-learn
- PyMuPDF

#  Features
- Upload Resume (PDF)
- Paste Job Description
- Calculate Resume Match Score
- Simple and Interactive UI

# Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
